## Setup

This repo doubles as my dotfiles and Mac dev environment setup:

```bash
# Install everything (Homebrew, CLI tools, apps, fonts)
bash scripts/setup-mac.sh

# Link dotfiles to home directory
bash scripts/install-dotfiles.sh
```

**What's included:**
- `scripts/setup-mac.sh` - Full Mac dev environment installer (idempotent)
- `scripts/install-dotfiles.sh` - Dotfile symlinker with auto-backup
- `dotfiles/` - zsh, git, vim, tmux, starship configs
